# What Is LangChain

## LangChain을 처음 보는 사람에게

LangChain은 LLM을 포함한 AI 애플리케이션을 구조적으로 만들도록 돕는 도구 모음입니다.

이 프로젝트에서는 다음 세 가지에 특히 중요합니다.

- 프롬프트 관리
- 도구 연결
- 에이전트 구조

## 이 프로젝트에서 LangChain이 하는 일

- `ChatPromptTemplate`로 프롬프트 구성
- `DynamicStructuredTool`로 tool 정의
- agent 계층에서 요청 해석과 실행 순서 조정

즉 "LLM 한 번 호출"보다 더 구조적인 앱을 만들게 도와줍니다.
