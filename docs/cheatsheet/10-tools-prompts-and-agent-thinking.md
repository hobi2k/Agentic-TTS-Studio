# Tools, Prompts, And Agent Thinking

## Prompt

Prompt는 모델에게 주는 지시문입니다.

이 프로젝트에서는 시스템 힌트와 사용자 메시지를 합쳐 prompt를 만듭니다.

## Tool

Tool은 모델이 직접 할 수 없는 일을 대신하는 함수입니다.

예:

- 로컬 모델 점검
- 음성 생성
- 프리셋 저장

## Agent thinking

이 프로젝트의 에이전트는 아래 질문을 스스로 묻는다고 생각하면 이해하기 쉽습니다.

1. 사용자가 지금 원하는 핵심은 무엇인가
2. 실제 작업이 필요한가
3. 어떤 tool을 써야 하는가
4. 결과를 어떻게 설명할까
